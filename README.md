Key Changes & Features:
Tabbed Interface: Added a sleek navigation bar to switch between the Crop Tool and the external Image Resizer.
Professional UI/UX:
Modern color palette with soft shadows and rounded corners.
Toast Notification System: Replaced native alert() boxes with elegant, non-intrusive toast popups.

External Integration: The Image Resizer is embedded via an iframe in the second tab, allowing you to use it without leaving the page.
